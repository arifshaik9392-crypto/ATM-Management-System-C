# ATM Management System in C

## 📌 About
This is a menu-driven ATM Management System developed using the C programming language.

## ✨ Features
- PIN Login
- Check Balance
- Deposit Money
- Withdraw Money
- Change PIN
- Logout

## 🛠️ Technologies Used
- C Programming
- Functions
- Loops
- Switch Statements

## 🚀 Future Improvements
- File Handling
- Transaction History
- Multiple User Accounts
